# Badge Icon Generation Prompts

**Naming convention:** `{group_slug}_{rank}.png`
**Storage:** Supabase Storage → `badge-icons` bucket (public)
**Size:** 512×512 PNG

Each rank has its own dark radial gradient background baked into the image. The CSS badge frame provides the border, glow, and animations — the image fills the circular frame.

---

## Background gradients per rank

| Rank | Background directive |
|---|---|
| Bronze | `on a dark radial gradient background from warm copper brown center to near-black edges` |
| Silver | `on a dark radial gradient background from cool steel grey center to near-black edges` |
| Gold | `on a dark radial gradient background from deep amber gold center to near-black edges` |
| Platinum | `on a dark radial gradient background from deep blue-slate center to near-black edges` |
| Diamond | `on a dark radial gradient background from deep purple indigo center to near-black edges` |

> **Suffix appended to every prompt:**
> `, centered composition, game UI icon asset, circular vignette, no border, no frame, no text, high detail, 512×512 PNG`
